Using Markdown.

Markdown is awesome. You can compose your messages using markdown in Noted. Here's a bit more about Markdown and why it rocks.

  • Markdown is simple to use and easy to learn. - Markdown is a easy to use and easy to learn markup language. Bloggers use it, many journalists use it and many developers use it to easily create something without having to worry about how something is done in HTML.
